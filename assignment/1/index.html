Lab Write-up: Static Site Generation (SSG) Using 11ty

Created on 11/23/2024, 12:46:32 AM

Nunjucks (we learned a bit in class) is also what we are using for our assignment.

Nunjucks is a templating engine, and it offers several options for templating, as we learned in class.

The Nunjucks environment can be customized. Eleventy allows us to provide our own instance of Nunjucks, giving us more control over the configuration.

Nunjucks provides several features:

Includes: Eleventy looks in the _includes directory, as seen in our class example.

Extends: allows for reusable layouts, also found in the _includes directory.

Imports: is used to reuse macros.

Filters: Allow content modification before rendering.

I learned that Nunjucks offers logic and better control, which helps to improve code organization.

Pros

- Nunjucks is a more modern template engine. It is flexible and secure.

- Components, shortcodes, and filters are reusable, which helps in organizing code. Additionally, filters enable data transformation and provide flexibility for real-time processing. Example: API integration.

- Flexibility: Nunjucks allows for custom environments, giving more control over configuration.

- We can add asynchronous filters and shortcodes to modify how content is rendered, providing good benefits for customization.

- Eleventy has universal filters and shortcodes, which makes Nunjucks code reusable in other languages.

Cons

- In my personal opinion, I find Nunjucks a bit complex. It can affect performance.

- It also seems to be a large template engine, which can impact performance.

Handlebars

- Handlebars is a templting use to create dynamic HTML. It separates logic from markup.

- It extends Mustache with features or helpers, it makes a versatil tool for front-end developers.

Pros

based on the information we have a few good befefits from Hanlebars such us:
