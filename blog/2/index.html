Integrating 11ty, Strapi, and React

by Inguer Lara

Written 11/22/2024, 7:26:20 PM

I was actually interested in learning a bit about "React", so here's a little explanation of my implementation:

Practical and efficient use of React to render dynamic content from Strapi in an 11ty project

Strapi provides structured content (e.g., assignments) via its API, which is then consumed and rendered using React components within an 11ty-generated static site.

React Component: Assignment.tsx

The Assignment component is responsible for rendering the assignment's details dynamically. It destructures data passed as props, including the title, body, and createdAt fields from the Assignment type. A RichTextBlock component handles rendering the body content, ensuring rich text is displayed appropriately.

Key highlights include:

Date Formatting: to make it move human re-readable.

Separation of Concerns: The RichTextBlock component encapsulates the logic for rendering the body, keeping the Assignment component clean and focused.

Type Definition: Assignment.ts

The Assignment type ensures type safety by defining the structure of the assignment object fetched from Strapi. It includes fields for metadata (id, createdAt, updatedAt, etc.) and the body, which is an array of RichTextBlock objects.
